# Learning\_GO



This repository collects my first steps learning \*\*Golang\*\*, with small exercises created to understand the basics of the language.



\## Objectives of this repository



\- Practice the fundamentals of the Go language.

\- Learn and reinforce:

&nbsp; - Variable declaration

&nbsp; - Basic data types

&nbsp; - Explicit type conversion

&nbsp; - String formatting and concatenation



Furthermore, the long‑term goal is to build a solid foundation for future \*\*cybersecurity‑related projects\*\* written in Go.
